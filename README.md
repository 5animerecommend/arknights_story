# Arknights_reader 1.0
작업은 Aceship, fexli, 050644zf, akgcc를 참조해서 이루어졌으며 비영리적으로 사용합니다.
해당 웹 페이지에 나오는 모든 텍스트, 이미지, 음원의 저작권은 게임 명일방주의 제작사인 하이퍼그리프에 있습니다.

아직 스토리만 볼 수 있습니다.



## 임시 실행 명령어

다음 항목을 실행할것

1. 터미널에서 python -m http.server 8000 실행
2. http://localhost:8000 접속
3. 종료시 Ctrl+C 또는 세션 닫기

## 업데이트 예정 항목
1. localstorage 활용 설정 저장해놓기
2. 설정에 따른 다이얼로그 변경
3. 모바일환경 생성
4. bgm 모음 및 음악 재생목록 생성
